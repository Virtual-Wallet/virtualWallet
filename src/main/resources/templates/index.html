Welcome to FLEX PAY

Your number one virtual wallet dynamic web application

Our Java project, named Flex Pay, is a virtual wallet app developed using Java, HTML/CSS, JavaScript, and the Spring framework. It allows users to manage their very own currency wallet, make secure transactions, based on real-time exchange rates.

Learn more about our cool features below. Happy Flexing!

Services

The features our application supports are: Email verification, Large transaction Verification, Identity Verification, Contacts List, Multiple Cards, Spending Categories, Currency support. Learn more for a of few of them below.

Email Verification

Ensure user account security and validity by requiring email verification during the registration process, reducing the risk of unauthorized access.

Large Transaction Verification

Implement an additional layer of security by requiring verification (such as a confirmation prompt or OTP) for large transactions to prevent fraudulent activities and protect user funds.

Identity Verification

Enable users to verify their identity using official documents or identity verification services to comply with regulatory requirements and enhance trust and security within the platform.

Contacts List

Allow users to create and manage a list of contacts (friends, family, merchants) within the app for quick and easy money transfers, improving user convenience and engagement.
